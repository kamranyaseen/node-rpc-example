# node-rpc-example
build an RPC based API with node.js

Service URL:
>http://localhost:9090/rpc.

Call the createUser method 
```javascript
{
    "createUser": {
        "name":"Kamran Yaseen",
        "age":21
    }
}
```


Call the fetchAllUsers method 
```javascript
{
	"fetchAllUsers":""
}
```


Call the fetchUser method 
```javascript
{
    "fetchUser": {
    	"id":"6699468"
    }
}
```
